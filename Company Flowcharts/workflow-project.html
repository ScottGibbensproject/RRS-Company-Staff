Important Notes

  • Agent Type Source: The feedback form at the end of certification should capture agent type (Direct Agent vs ISO Group Agent)
  • ISO Group Assignment: ISO Group Agents need to be linked to their specific ISO group for proper commission tracking
  • Email Content: Direct Agent emails are RRS-centric; ISO emails should acknowledge the ISO relationship
  • Support Escalation: ISO Agents should contact their ISO first; Direct Agents contact RRS directly
  • Future Consideration: May need separate workflows for different ISO Groups if they have unique requirements
