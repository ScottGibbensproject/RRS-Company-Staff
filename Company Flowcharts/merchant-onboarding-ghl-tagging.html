Merchant Tagging Logic Rules

RRS - Auto RepairNEW

Condition: Business Industry = Auto Repair

Applied: All merchants in auto repair industry, regardless of reseller type.

HouseNEW

Condition: Reseller Code = null (empty)

Applied: Merchants with no sales rep / no reseller code entered at checkout.

RRS Direct AgentNEW

Condition: Reseller Code starts with RTR

Applied: Merchants signed by Direct Agents working under House Account.

RRS ISO GroupNEW

Condition: Reseller Code exists AND does NOT start with RTR

Applied: Merchants from ISO, ISO Agent, or Referral Partner accounts.
