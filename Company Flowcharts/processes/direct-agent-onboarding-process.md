# Direct Agent Onboarding Process

## Overview
This document outlines the process for onboarding a new Software Reseller - Direct Agent.

---

## Trigger
**Agent makes payment for setup fee**

- Add tag in GHL: `Direct Agent Setup Fee Received`
- GHL tag triggers Workflow: **"Direct Agent Onboarding"**

---

## Workflow Tasks

| Task # | Description | Assigned To | Due |
|--------|-------------|-------------|-----|
| 1 | Send Subagent Pack to EPI | Nicole | 1 day |
| 2 | Automatic email: "Welcome to Rate Remover Software! Here's what's next" | Automatic | Immediate |
| 3 | Create Merchant Demo Account (Follow SOP in Google Sheet) | Elisabeth | 2 days |
| 4 | Send Direct Agent Starter Pack (Follow SOP) | Elisabeth | 2 days |
| 5 | Order Business Cards | Elisabeth | 2 days |
| 6 | Complete Onboarding Call | Nicole | 7 days |

---

## Task Details

### Task 1: Send Subagent Pack to EPI
- **Assigned to:** Nicole
- **Due:** 1 day after trigger
- **Action:** Send the Subagent Pack to EPI

### Task 2: Welcome Email (Automatic)
- **Type:** Automated
- **Timing:** Immediate
- **Email Subject:** "Welcome to Rate Remover Software! Here's what's next"

### Task 3: Create Merchant Demo Account
- **Assigned to:** Elisabeth
- **Due:** 2 days after trigger
- **Reference:** Follow SOP in Google Sheet

### Task 4: Send Direct Agent Starter Pack
- **Assigned to:** Elisabeth
- **Due:** 2 days after trigger
- **Reference:** Follow SOP

### Task 5: Order Business Cards
- **Assigned to:** Elisabeth
- **Due:** 2 days after trigger

### Task 6: Complete Onboarding Call
- **Assigned to:** Nicole
- **Due:** 7 days after trigger
- **Purpose:** Final onboarding review with new Direct Agent
